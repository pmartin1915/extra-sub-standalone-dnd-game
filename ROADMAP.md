# Roadmap -- extra-sub-standalone-dnd-game

## Phase 1: Core Mechanics Design (cycles 1-4)
- [ ] Design dice system (d4, d6, d8, d10, d12, d20, percentile)
- [ ] Design ability scores and modifiers (STR/DEX/CON/INT/WIS/CHA)
- [ ] Design combat resolution (attack roll vs AC, damage, critical hits)
- [ ] Design character creation (race, class, stats, starting equipment)
- [ ] Document mechanics in a design doc

## Phase 2: Engine Foundation (cycles 5-10)
- [ ] Implement dice roller with modifier system
- [ ] Implement character model (stats, HP, AC, equipment slots)
- [ ] Implement basic attack/damage resolution
- [ ] Implement teaching toggle (verbose vs terse output)
- [ ] Unit tests for all combat math

## Phase 3: Character & Combat (cycles 11-18)
- [ ] Character creation flow (interactive terminal)
- [ ] Initiative and turn order
- [ ] Multiple enemy types with challenge ratings
- [ ] Encounter difficulty scaling
- [ ] Saving throws and conditions
- [ ] Death saves

## Phase 4: Progression & Content (cycles 19-30)
- [ ] Experience and leveling system
- [ ] Equipment and inventory management
- [ ] Multiple character classes with distinct abilities
- [ ] Encounter/adventure generator
- [ ] Lore adapter interface for worldbuilder integration

## Phase 5: Polish (cycles 31+)
- [ ] Game balance review and tuning
- [ ] Web UI (optional upgrade from terminal)
- [ ] Teaching toggle UX refinement
- [ ] Comprehensive documentation
